In this Nation Server, when someone wants to begin a war they have to do that using Casus-Belli requests (CBs) and War requests.
Currently the only way to create CB requests is using the discord server. But eventually I plan on creating a better way without the discord server.

  • Linking your Discord to your Minecraft username to use the Discord server
  • Making a CB request.
  • Making a War request

Linking your Discord to your Minecraft username to use the Discord server

Before making a request using discord you will need to link your Discord to your minecraft username on the server. It's a very simple and quick process.

  1. First join the discord server using this invite link
  2. Then run the command "/discord link",
    • it should give you a string of numbers, note the numbers down.
  3. Go back to the discord server, look in the members list and open Direct messages with the "silly SRV" bot.
  4. Send the string of numbers that "/discord link" gave you in step 2.

Now you should be able to see all the channels!

Making a CB request.

  1. To start maing a CB Request, first navigate to the "cb-requests" channel.
  2. Start writing out a new message with this format
    [Your nation] Nation name
    [Enemy nation] Enemy nation name
    [Casus-Belli] Casus Belli type
    [Evidence] Can be an image or whatever to prove that the enemy nation did that CB
    • You can create new lines by doing shift+enter.
  3. Replace "Nation name" with the name of your nation.
  4. Replace "Enemy nation name" with the name of the nation you are making this cb request on.
    • You can do /f who <username> (Replacing "<username>" with the persons username) to find the name of the nation that person is in.
  5. Replace "Casus Belli type" with the type of casus belli you're doing.
    • You can find a list of these in the rules page
  6. Replace "Can be an image or whatever to prove that the enemy nation did that CB" with whatever evidence you have to justify the Casus Belli request.
    This can be something like
    • A message link to a message in the "ingame-chat" channel.
    • An image with proof it's from the server.
    • The time and place of when an event happened. (Only if it can be realisticly found from the logs, such as a block break, or entity kill.)
  7. Once you've filled out all the neccessary stuff, just send the message and wait for a staff member to see it.

Once a staff member sees the CB request, they'll send a message saying whether it was accepted or denied, and sometimes a thread may be made to discuss some things before it's accepted or denied.
Once it's accepted you have usually 2 weeks to make a war request using that CB.

Making a War Request

  1. To start making a War request, first navigate to the "wars" channel.
  2. Start writing out a new message with this format
    [Your nation] Nation name
    [Enemy nation] Enemy nation name
    [Casus-Belli used] Message link to cb request
  3. Replace "Nation name" with the name of your nation.
  4. Replace "Enemy nation name" with the name of the nation you are making this cb request on.
  5. Replace "Message link to cb request" with a message link to the accepted CB Request you are using to justify this war.
    • You can grab the message link by right clicking or clicking the "More" button when hovering over a message, and selecting the "Copy Message Link" button.
  6. Once you've filled out all the neccessary stuff, just send the message and wait for a staff member to see it.
  7. Once a staff member accepts the War request, a thread will be made on the channel, where you will need to discuss when the war will start.
  8. After the war is ended, in that same thread the winning nation will have to send all their post-war actions, once they are checked and accepted by a staff member, the losing nation will have to follow along with those post-war actions.
